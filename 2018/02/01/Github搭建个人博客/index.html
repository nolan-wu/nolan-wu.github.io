hexo常用命令

  • hexo init #初始化
  • hexo clean #清理临时文件
  • hexo g # 完整命令为hexo generate，生成静态文件
  • hexo s #完整命令为hexo server，打开本地服务
  • hexo d #完整命令为hexo deploy，用于将本地文件发布到github上
  • hexo new “postName” #新建文章
  • hexo new page “pageName” #新建页面
  • hexo help #帮助
  • hexo d -g #同步至github
