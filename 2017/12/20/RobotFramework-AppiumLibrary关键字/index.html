1.条件判断函数

关键字：Run Keyword If

用法：
Run Keyword If
…ELSE IF
…ELSE

实例：
1）用Run Keyword If执行单条语句

2）用Run Keyword If执行多条语句时，在需要执行多条语句的if或else if或else后，增加关键字：Run Keywords

2.检查某关键字的返回状态

关键字：Run Keyword And Return Status
返回值：布尔值，True False

实例：通常和Run Keyword If关键字搭配使用

含义：判断当前界面中是否存在取消按钮；如果存在，则do something
